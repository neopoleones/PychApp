class Chat:

View In Hierarchy

A class representing a chat session between users.

This class handles the initialization of a chat session, managing encryption and user details. It can operate in two modes: encrypted and plain. In encrypted mode, it uses RSA encryption for securing the chat. In plain mode, it simply stores the given secret as is for use in api endpoints.

def __init__(self, secret, aes, u_init, u_dest, cid=None, plain=False): ¶

Initialize a Chat instance.

In encrypted mode, 'u_init' and 'u_dest' should be User objects with RSA keys. In plain mode, they should be strings representing user logins. The AES key is decrypted in encrypted mode.

Parameters
secretThe secret key for RSA encryption/decryption.
aesThe AES key, either encrypted or plain.
u_initThe initiating user's object or login, depending on the mode.
u_destThe destination user's object or login, depending on the mode.
cidAn identifier for the chat session. Defaults to None.
plainA flag to indicate if the chat is in plain mode (not encrypted). Defaults to False.
